# Car Wash Management System

A Java **Servlet-based** internal management system for handling daily car wash operations.  
The project uses **pure HTML pages + Servlets + MySQL**, with **no JSP** and **no MVC architecture**.  
All customer work entries are added by **Admin or Staff only**. Customers do **not** access the system.
